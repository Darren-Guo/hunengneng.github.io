使用 Hugo 搭建个人博客#

今天我想分享一下如何使用 Hugo 这个强大的静态网站生成器来搭建个人博客。

为什么选择 Hugo？#

Hugo 是一个用 Go 语言编写的静态网站生成器，具有以下优点：

  1. 速度快: 构建速度极快，即使是大型网站也能在几秒内完成
  2. 简单易用: 命令行工具简单直观，学习曲线平缓
  3. 主题丰富: 有大量高质量的主题可供选择
  4. 部署方便: 可以轻松部署到 GitHub Pages、Netlify 等平台

安装 Hugo#

macOS#

brew install hugo

Ubuntu/Debian#

sudo apt-get install hugo

Windows#

choco install hugo-extended

创建新网站#

hugo new site my-blog
cd my-blog

添加主题#

我选择了 PaperMod 主题，因为它简洁美观且功能丰富：

git init
git submodule add https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod

配置网站#

编辑 hugo.toml 配置文件：

baseURL = 'https://yourusername.github.io/'
languageCode = 'zh-cn'
title = '我的个人博客'
theme = 'PaperMod'

[params]
  homeInfoParams = 
    Title = "欢迎来到我的博客"
    Content = "这里分享技术学习和生活感悟"

创建内容#

创建第一篇文章：

hugo new content posts/first-post.md

本地预览#

启动本地服务器：

hugo server -D

然后在浏览器中访问 http://localhost:1313 查看效果。

部署到 GitHub Pages#

  1. 在 GitHub 上创建名为 yourusername.github.io 的仓库
  2. 构建静态文件： 
    hugo
  3. 将 public 目录的内容推送到仓库

总结#

Hugo 是一个优秀的静态网站生成器，特别适合技术博客和个人网站。它的速度快、配置简单，而且有丰富的主题生态系统。
